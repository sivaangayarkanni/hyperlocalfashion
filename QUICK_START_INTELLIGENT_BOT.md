# 🚀 Quick Start - Intelligent Chatbot

## ⚡ 30-Second Start

```bash
# Start the application
npm run dev

# Open browser
http://localhost:3000

# Click chat button (bottom right) 💬

# Try: "Find me a tailor for my torn jeans"
```

**That's it! Your intelligent bot is ready! 🎉**

---

## 🧠 What Makes It Intelligent?

### 1. **Understands Intent** (9 types)
```
"Find a tailor and tell me the price"
→ Detects BOTH intents and responds to both!
```

### 2. **Extracts Information**
```
"Urgent repair for torn jeans under ₹500"
→ Extracts: jeans, torn, urgent, ₹500
```

### 3. **Detects Emotions**
```
"This is terrible!" → Empathetic response
"Amazing service!" → Celebratory response
```

### 4. **Remembers Context**
```
You: "Find tailors"
Bot: [Shows 3 tailors]
You: "Tell me about the second one"
Bot: [Knows which tailor you mean]
```

### 5. **Personalizes Everything**
```
"Hi John! With 5 bookings and 12.5kg CO2 saved,
you're a sustainability champion! 🌱"
```
