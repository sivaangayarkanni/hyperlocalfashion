## 🚀 HOW TO DEPLOY

### Option 1: Quick Docker (Recommended)
```bash
git clone https://github.com/sivaangayarkanni/hyperlocalfashion.git
cd hyperlocalfashion
docker-compose up -d
# Access: http://localhost:3000
```

### Option 2: Automated Script
```bash
# Linux/Mac
./deploy.sh

# Windows
deploy.bat
```

### Option 3: Manual
```bash
docker build -t rewear-platform:latest .
docker run -d -p 5000:5000 -p 3000:3000 rewear-platform:latest
```
